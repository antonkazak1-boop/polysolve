# Тестовые «семена» для MiroFish (Polymarket-сценарий)

**Важно:** всё ниже — **вымышленный сценарий** для проверки движка. Не используйте как финансовый совет и не путайте с реальными рынками.

---

## Как использовать

1. Скопируйте в MiroFish блок **«Данные (вставить целиком)»**.
2. В поле запроса вставьте **«Вопрос для симуляции»** (ниже отдельным блоком).
3. Для экономии токенов в их UI начните с **короткой** симуляции (мало раундов), как советуют в README MiroFish.

---

## Вопрос для симуляции (вставить в промпт)

```
Ты анализируешь вымышленное событие на prediction market (аналог Polymarket).

Задача:
1) Исходя только из приложенных «семян» (новости, логи, таймлайн), какие 3–5 сценариев развития наиболее правдоподобны до дедлайна?
2) Для каждого сценария: ключевые триггеры, кто выигрывает/проигрывает внимание рынка, что должно произойти в реальном мире, чтобы сценарий сбылся.
3) Какие факты в данных противоречат друг другу и создают неопределённость?
4) Если бы ты был риск-менеджером с лимитом экспозиции, какие правила стопа/пересмотра гипотезы ты бы задал (без конкретных сумм)?

Ответ структурируй заголовками и маркированными списками, без выдумывания внешних источников — только интерпретация приложенного текста.
```
